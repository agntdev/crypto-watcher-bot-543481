# Crypto Watcher Bot

Telegram bot for private, low-noise crypto price monitoring with watchlists, threshold and percent-move alerts, morning summaries, quiet hours and cooldowns.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
